# SISTEMA DE SEGUIMIENTO DE DONACIONES

Dashboard interactivo con KPIs de estado y Diagrama de Gantt para gestionar el avance de una campana de donaciones.

## Caracteristicas
- Calculos automaticos de porcentajes de avance
- Tarjetas visuales de KPIs con colores de estado
- Diagrama de Gantt completo (Enero - Diciembre)
- Diseno responsivo (funciona en PC, tablet y celular)
- 100% funcional sin instalacion ni internet

## Tecnologias
- HTML5 - Estructura del sistema
- CSS3 - Diseno y estilos visuales
- JavaScript - Calculos dinamicos y logica

## Inicio Rapido
Solo abre el archivo index.html en cualquier navegador web.
