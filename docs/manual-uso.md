# MANUAL DE USO - SISTEMA DE DONACIONES

## Como ejecutar el sistema
1. Descomprime el archivo ZIP en tu computadora
2. Abre la carpeta sistema-donaciones
3. Haz doble clic en el archivo index.html
4. Se abrira automaticamente en tu navegador web

No requiere instalacion, servidor ni internet. Funciona en cualquier navegador moderno.

## Como modificar los datos
Para cambiar valores (cantidad de donaciones, fechas, etc.):
1. Abre la carpeta js
2. Edita el archivo logica.js con cualquier editor de texto
3. Modifica las variables total y realizadas segun tus nuevos datos
4. Guarda y actualiza la pagina del navegador

## Como cambiar colores y diseno
1. Abre la carpeta css
2. Edita el archivo estilos.css
3. Modifica los valores de colores

## Estructura del sistema
sistema-donaciones/
- index.html          Pagina principal
- css/estilos.css     Diseno y colores
- js/logica.js        Calculos y datos
- assets/             Recursos graficos
- docs/               Documentacion
- README.md           Informacion general
